# CSharp Projects

This repository contains beginner-level C# console applications created using Visual Studio.

## Projects Included

- Number Comparison Program  
  Checks whether two numbers entered by the user are equal or not.

- Sum of First 10 Natural Numbers  
  Prints the first 10 natural numbers and calculates their total sum.

- Menu Driven Calculator  
  A calculator program that performs addition, subtraction, multiplication, and division using a menu system.

## Skills Practiced

- C# basics  
- Console input and output  
- Variables and data types  
- Loops  
- Conditional statements  
- Switch-case logic  

## Tools Used

- Visual Studio  
- C# (.NET Console Applications)
